# NutriMind Admin Web

`admin-web` is the Vue 3 web console for NutriMind.

## Planned Admin Scenarios

- Knowledge base maintenance for RAG content
- Food catalog maintenance
- Community moderation
- System status and model readiness overview
- Operations dashboard for contest demos

## Local Development

```bash
cd admin-web
npm install
npm run dev
```

By default the app points to `http://localhost:8080/api`.
You can change the backend target in the system settings page after startup.
